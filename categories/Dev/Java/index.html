카이 호스트만의 코어 자바8

카이 호스트만의 코어 자바8

책 소개

나는 Java 를 배웠지만 내가 Java 를 제대로 배웠다고 생각하지 않는다.
중간중간 이런 저런 일이 많아서 끊기면서 배웠고, 실제로 자주 사용하지 않아 많이 잊어 버렸다.
그래서 요즘 핫한 Java8 을 배울겸 이 책을 구매하게 되었다.
기존에는 Java의 정석으로 보았으나, Java8 에 대한 내용이 없기 때문에 이 책과 병행하여 보도록 할 것이다.
주로 정석 보다는 이 책을 보게 될 것이며, 보면서 내가 정리해야겠다고 느끼는 부분들을 정리 할 것이다.
책은 Java 를 처음 배우는 사람보다는 기본적인것은 알고 있다는 전제를 가지고 기술하는 느낌이다.
이론적인 내용보다는 실용적이고 실무적인 내용을 기술한다고 되어 있는데 아직 많이 보지 않아서 잘 모르겠다.
책을 보며 느끼는 바가 생기면 바로바로 업데이트를 하도록 하겠다.
